One-time setup

Deploy a free Google Apps Script web app to enable saving data to your sheet.

  1. Open your Google Sheet
  2. Click Extensions → Apps Script
  3. Delete all existing code and paste the script below
  4. Click Deploy → New deployment → Web app
  5. Execute as: Me  ·  Who has access: Anyone
  6. Authorize, then copy the Web App URL and paste below
